## ğŸ”„ Application Workflow (Device Usage)

1. Install the demo APK on a supported Android device.
2. Launch the application and navigate to **New Capture**.
3. Align your fingertips within the guide box and follow the on-screen alignment prompts.
4. Hold the device steady until automatic capture completes, or manually trigger capture using the capture button.
5. Open **Enhance** to review:
   - Masked raw image  
   - Enhanced masked image  
   - Skeleton output
6. Use **Save in ISO Format** to export the skeleton image in **PNG** or **TIFF** format, if required.
7. Navigate to **Match** to compare the most recent capture against candidate samples.
8. Open **Settings** to configure options such as auto-capture and other preferences.
